## Current State (Pre-Convex)
- `convex/_generated/` directory exists but only contains `README.md`
- No `api.d.ts` or `dataModel.d.ts` files present
- These files are auto-generated by `npx convex codegen` after the Convex backend is initialized and running

## Required Commands to Generate Types
1. Start Convex development server:
   ```bash
   npx convex dev
   ```
   (This will prompt for project setup if not already configured)

2. In a separate terminal, generate TypeScript types:
   ```bash
   npx convex codegen
   ```

3. Verify generated files exist:
   ```bash
   ls -la convex/_generated/
   # Should show api.d.ts, dataModel.d.ts, etc.
   ```

- `npx vite build` will fail until types are generated
- These are **not blocking errors** — they resolve automatically after running `npx convex codegen`
