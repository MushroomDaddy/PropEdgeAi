### 9. Exact Convex go-live command sequence
```bash
# 1. Install dependencies (npm ci requires package-lock.json)
cd /tmp/PropEdgeAi
npm ci

# 2. Login to Convex (first time only)
npx convex login

# 3. Initialize/link project (first time only)
npx convex init

# 4. Start Convex dev server (generates types, watches functions)
npx convex dev
# Leave this running in one terminal

# 5. In another terminal, generate Convex types
npx convex codegen

# 6. After codegen, rerun type checks / build
npx tsc -p tsconfig.app.json --noEmit
npx tsc -p convex/tsconfig.json --noEmit
npm run build

# 7. Set environment variables in Convex Dashboard
# Go to: https://dashboard.convex.dev → Your Project → Settings → Environment Variables
# Add:
#   VITE_CONVEX_URL (also in .env.local for frontend)
#   THE_ODDS_API_KEY
#   API_SPORTS_KEY
#   THESPORTSDB_API_KEY (free dev key: 123)
#   BALLDONTLIE_API_KEY
#   KALSHI_API_KEY
#   SERPAPI_KEY
#   OPENAI_API_KEY
#   ADMIN_EMAILS (comma-separated emails)
#   SMOKE_TEST_TOKEN (for goLiveSmokeTest)

# 8. Run goLiveSmokeTest (requires SMOKE_TEST_TOKEN)
npx convex run goLiveSmokeTest:run '{"token":"YOUR_TOKEN","sport":"NBA"}'

# 9. Seed demo data (optional)
npx convex run seed:clearAndReseed
npx convex run seedR5:seedGameDetails

# 10. Start dev server
npm run dev
```
